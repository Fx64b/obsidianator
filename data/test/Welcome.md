# Welcome to Obsidianator

**Obsidianator** is a tool that exports your [Obsidian](https://obsidian.md) vault to a self-contained static website.

## What it does

- Parses all your `.md` notes
- Resolves `[[wikilinks]]` between notes
- Extracts tags, headers, and frontmatter
- Renders a fast, searchable web UI

## Getting Started

See [[Getting Started]] for instructions on how to use Obsidianator.

For a full list of supported features, check out [[Features]].

## Why Obsidianator?

Your notes deserve to be shared. Obsidianator turns your vault into a self-contained website you can host anywhere — no server required, no Node.js at runtime.
