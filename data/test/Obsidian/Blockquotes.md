# Blockquotes

Standard markdown blockquotes (distinct from callouts).

## Simple Blockquote

> This is a simple blockquote.

> "The best way to predict the future is to invent it."
> — Alan Kay

## Multi-Paragraph Blockquote

> This is the first paragraph of the blockquote. It contains several sentences to make the paragraph a reasonable length.
>
> This is the second paragraph inside the same blockquote. A blank `>` line separates paragraphs within a blockquote.

## Nested Blockquotes

> This is an outer blockquote.
>
> > This is a nested blockquote inside the outer one.
> >
> > > This is triply nested.

## Blockquote with Markdown

> **Bold text** and *italic text* inside a blockquote.
>
> - List item one
> - List item two
> - List item three

## Blockquote with Code

> Here's an example command:
>
> ```bash
> obsidianator export ./vault --output ./dist
> ```
>
> Run it from your project root.

## Blockquote with Heading

> ### Heading Inside Blockquote
>
> Content following the heading inside the same blockquote.

## Long Quoted Passage

> In the beginning God created the heaven and the earth. And the earth was without form, and void; and darkness was upon the face of the deep. And the Spirit of God moved upon the face of the waters.
>
> And God said, Let there be light: and there was light.
