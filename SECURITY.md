# Security Policy

## Supported Versions

Only the latest release is actively maintained and receives security fixes.

## Reporting a Vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Report security issues by emailing **obsidianator.discourse992@passmail.net** with:
- A description of the vulnerability
- Steps to reproduce it
- The potential impact

You will receive a response within 7 days. Once the issue is confirmed and a fix is released, you will be credited in the release notes (unless you prefer to remain anonymous).
